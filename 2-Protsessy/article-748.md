# Что такое снапшоты?

**Что такое снапшоты?**

- техника сохранения **состояния** системы/данных в момент времени  
- используется для быстрого восстановления без полной переинициализации  
- в БД — копия данных для точной точке времени (point-in-time)  
- в виртуализации — снимок состояния ВМ для отката  
- минимизирует время отката и потери данных  
- часто инкрементальные, экономят место и ресурсы  
- важны для резервного копирования и восстановления в продакшн  

Итог: снапшоты — это эффективный способ зафиксировать состояние для быстрой откатки или анализа.

# Подробный ответ
## Основной ответ
**Снапшоты** — это точечные копии состояния системы, базы данных или виртуальной машины в определённый момент времени, которые позволяют быстро восстановить или проанализировать состояние без необходимости восстанавливать данные по отдельным изменениям. Снапшоты часто используются для создания резервных копий, точек отката или для клонирования среды.

## Ключевые моменты
- **Консистентность данных**: снапшот фиксирует единое согласованное состояние, что важно для баз данных и файловых систем, чтобы избежать частичных искажений данных.
- **Copy-on-write или redirect-on-write**: современные технологии снапшотов (например, LVM, ZFS) обычно используют эти техники для эффективного хранения, сохраняя только изменения после создания снапшота, что экономит дисковое пространство.
- **Производительность и время создания**: снапшоты создаются быстро (обычно за миллисекунды) без полной копии данных, что позволяет не останавливать работу приложений, но при частом использовании могут влиять на производительность.

## Практический контекст
В VMware ESXi снапшоты сохраняют состояние виртуальной машины, чтобы можно было откатиться к предыдущему состоянию в случае проблем. В СУБД, например, PostgreSQL использует снимки для реализации MVCC (многоверсионная конкурентность), чтобы обеспечить согласованное чтение без блокировок. В Kubernetes снапшоты томов Persistent Volume позволяют создавать резервные копии или мигрировать данные без остановки подов.
