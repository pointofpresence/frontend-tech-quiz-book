# Что такое дженерики? Приведите пример использования дженериков.

Категория: Процессы

### Краткий ответ

**Что такое дженерики? Пример использования дженериков**

- дженерики — механизм **типизации** для создания **универсальных** компонентов  
- позволяют писать **функции, классы, интерфейсы** с параметрами типов  
- обеспечивают **типобезопасность** без потери гибкости  
- компилятор подставляет конкретный тип при использовании  
- пример: функция **swap<T>(a: T, b: T): [T, T]** меняет местами значения любого типа  
- популярно в языках: **Java, C#, TypeScript**  
- применяются для коллекций, утилитных функций, API с универсальной логикой  

Пример (TypeScript):  
```typescript
function identity<T>(arg: T): T {
  return arg;
}
let output = identity<string>("строка");
```  
— универсальная функция возвращает аргумент того же типа, что приняли — типобезопасно и переиспользуемо.

### Подробный ответ
#### Основной ответ
**Дженерики** — это механизм, позволяющий создавать обобщённые структуры данных и функции, которые могут работать с разными типами данных, сохраняя при этом типовую безопасность. Они позволяют параметризовать типы, что обеспечивает повторное использование кода, минимизируя дублирование и ошибки времени выполнения.

#### Ключевые моменты
- В языках как **Java** (начиная с Java 5) или **TypeScript** дженерики позволяют определить класс, интерфейс или метод с параметром типа, например `<T>`, который подставляется при создании экземпляра или вызове.
- Это улучшает читаемость и безопасность кода: компилятор проверяет корректность типов без кастов и предупреждает ошибки на этапе компиляции.
- Дженерики поддерживают ограничение типа (bounded generics), позволяя указать требования к типу, например `T extends Number` в Java для ограничения числовыми типами.

#### Практический контекст
Пример на Java — универсальный класс контейнера:
```java
public class Box<T> {
    private T value;
    public void set(T value) { this.value = value; }
    public T get() { return value; }
}
```
Используется так:
```java
Box<String> stringBox = new Box<>();
stringBox.set("Привет");
String greeting = stringBox.get();
```
Это позволяет создавать один класс `Box`, который будет работать как с `String`, так и с любыми другими типами, сохраняя безопасность типов.

В реальных проектах дженерики широко применяются в коллекциях (List<T>, Map<K,V>), в ORM для универсальных DAO, а также в функциональных интерфейсах и шаблонах проектирования для повышения гибкости кода.
