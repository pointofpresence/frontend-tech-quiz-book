# Каковы три основных типа коллекций в Swift?

Категория: Процессы

### Краткий ответ

**Три основных типа коллекций в Swift**  
- Swift имеет 3 базовые коллекции: **Array**, **Set**, **Dictionary**  
- **Array** — упорядоченный список, позволяет хранить дубликаты  
- **Set** — неупорядоченный набор уникальных элементов, основан на хешировании  
- **Dictionary** — коллекция пар ключ-значение, обеспечивает быстрый доступ по ключу  
- Все коллекции — **generic**, типобезопасные и изменяемые (var) или неизменяемые (let)  
- Используются для структурирования и эффективного поиска/сортировки данных в приложениях Swift

### Подробный ответ
#### Основной ответ
В **Swift** три основных типа коллекций — это **Array**, **Set** и **Dictionary**. Они обеспечивают разные структуры данных и используются для хранения и управления наборами элементов с различной логикой организации и уникальными возможностями.

#### Ключевые моменты
- **Array** — упорядоченный список элементов одного типа, доступ к которым осуществляется по индексу. Поддерживает операции вставки, удаления и сортировки. В Swift 5+ реализован с оптимизированным управлением памятью и copy-on-write механизмом для эффективности.
- **Set** — неупорядоченная коллекция уникальных элементов, тоже одного типа. Идеальна для операций, связанных с уникальностью, проверкой вхождения и множествами (пересечение, объединение). Использует хэш-таблицу, поэтому операции поиска работают в среднем за O(1).
- **Dictionary** — коллекция пар "ключ-значение", где ключ должен быть Hashable, а значения могут быть любого типа. Используется для быстрого доступа к значениям по уникальному ключу. Также реализован с высокой производительностью и copy-on-write семантикой.

#### Практический контекст
В реальных iOS-проектах **Array** применяют для упорядоченных списков, например, список записей. **Set** хорошо подходит для фильтрации дубликатов и проверки пересечений. **Dictionary** незаменим для отображения параметров или кэшированных данных с быстрым поиском по ключу. В Swift 5+ все эти коллекции поддерживают мощные функции, включая расширения, функциональные методы (`map`, `filter`, `reduce`) и работают эффективно благодаря современным оптимизациям компилятора.
