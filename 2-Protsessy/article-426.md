# Что такое псевдоклассы?

Категория: Процессы

### Краткий ответ

**Что такое псевдоклассы?**  
- CSS-специфика, часть **селекторов**  
- обозначают **состояния** элементов без изменения HTML  
- начинаются с двоеточия (:hover, :focus)  
- позволяют стилизовать, например: наведение, фокус, первые/последние элементы  
- работают динамически, управляют **интерактивностью** UI  
- расширяют возможности базовых селекторов  
- ключевы для адаптивного и интерактивного дизайна страниц

### Подробный ответ
#### Основной ответ
**Псевдоклассы** в CSS — это специальные селекторы, позволяющие применять стили к элементам на основании их состояния, позиции или отношений в DOM, без изменения самого HTML. Они начинаются с двоеточия `:` и помогают таргетировать элементы, например, при наведении курсора, первом дочернем элементе или элементе в фокусе.

#### Ключевые моменты
- **Динамические состояния**: псевдоклассы, такие как `:hover`, `:focus`, `:active`, позволяют стилизовать элемент при взаимодействиях пользователя, например, при наведении мыши.
- **Структурные псевдоклассы**: `:first-child`, `:nth-child()`, `:last-of-type` дают возможность выбирать элементы по их положению в HTML структуре без добавления классов.
- **Отличие от псевдоэлементов**: псевдокласс влияет на реальный элемент в DOM и его состояние, а псевдоэлемент (`::before`, `::after`) создаёт виртуальные части внутри элемента.
- В современных стандартах селекторы могут быть составными, например, `a:hover::after`.
  
#### Практический контекст
В реальных проектах псевдоклассы активно используют для создания интерактивных интерфейсов без JavaScript — например, меняют цвет кнопки при наведении (`button:hover`), выделяют активные элементы меню (`.nav-item:active`), или формируют стили для четных/нечетных строк таблиц (`tr:nth-child(even)`). Это облегчает поддержку кода и снижает количество скриптов.
