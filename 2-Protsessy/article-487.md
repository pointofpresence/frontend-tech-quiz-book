# Что такое ссылочный тип?

Категория: Процессы

### Краткий ответ

**Что такое ссылочный тип?**

- тип данных, хранящий **ссылку** на объект
- значение – адрес в памяти, не данные напрямую
- при присваивании копируется **ссылка**, не объект
- изменения объекта отражаются везде, где есть ссылка
- типичные примеры: **объекты**, массивы, функции (в JS)
- противоположность – **примитивные** типы, хранящие данные напрямую
- важно для управления памятью и передачи параметров функций

### Подробный ответ
#### Основной ответ
**Ссылочный тип** в программировании — это тип данных, переменная которого хранит не сами данные, а ссылку (адрес) на место в памяти, где эти данные реально лежат. В отличие от **примитивных типов**, которые содержат сами значения, ссылочные типы позволяют работать с большими или сложными структурами, передавая указатель на них, а не копируя полностью. Это важно для оптимизации памяти и производительности.

#### Ключевые моменты
- В языках как Java, C#, JavaScript объекты, массивы, функции — примеры ссылочных типов, тогда как числа, булевы значения — примитивы.
- При передаче ссылочного типа в функцию копируется ссылка, а не сам объект, поэтому изменения внутри функции повлияют на исходный объект.
- Ссылочные типы требуют управление памятью: в Java и C# — сборщик мусора, в C++ — умные указатели или ручное управление, чтобы избежать утечек.

#### Практический контекст
В React 18 и Node.js, где объекты и массивы — ссылочные типы, типично оптимизировать изменения состояния с учетом ссылочной идентичности для правильного рендера. Также в системах с большим объемом данных ссылки помогают избежать излишнего копирования и уменьшить задержки.
