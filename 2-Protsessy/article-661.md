# Как создать частичный индекс (условия, синтаксис)?

Категория: Процессы

### Краткий ответ

**Как создать частичный индекс (условия, синтаксис)?**

- частичный индекс — индекс на часть строк по условию  
- фильтрует данные по **WHERE** при создании индекса  
- синтаксис (PostgreSQL):  
  `CREATE INDEX имя_индекса ON таблица(колонка) WHERE условие;`  
- улучшает производительность запросов с фильтром, снижая размер индекса  
- применим, когда только часть строк активно используется в запросах  
- экономит ресурсы, ускоряет выборки с заданным условием  
- пример:  
  `CREATE INDEX idx_active ON users(email) WHERE active = true;`  
— индексирует только активных пользователей для быстрых запросов по email

### Подробный ответ
#### Основной ответ
**Частичный индекс** в PostgreSQL — это индекс, созданный только для подмножества строк таблицы, удовлетворяющих определённому условию. Он позволяет сократить размер индекса и ускорить запросы, которые работают с этим подмножеством данных. Синтаксис создания частичного индекса выглядит так:

```sql
CREATE INDEX имя_индекса ON имя_таблицы (колонка)
WHERE условие;
```

Условие после `WHERE` ограничивает строки, которые будут индексироваться.

#### Ключевые моменты
- **Повышение производительности**: Частичный индекс работает быстрее и занимает меньше места, если запросы часто фильтруют по условию из индексa.
- **Пример условия**: можно индексировать только «активные» записи — например, `WHERE status = 'active'`.
- **Поддержка во всех типах индексов**: можно создавать частичные B-tree, GiST, GIN, BRIN и другие индексы.
- **Использование в планировщике**: PostgreSQL автоматически выбирает частичный индекс, если условие в запросе покрывает условие индекса.

#### Практический контекст
Типично используют, когда в таблице большой объём данных, но большинство запросов затрагивает только часть строк, например, только текущие активные пользователи, а не все архивные. В PostgreSQL 14+ частичные индексы поддерживаются в дополнение к продвинутым возможностям обхода индексов для оптимизации запросов.
