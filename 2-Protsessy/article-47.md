# Как работает type guards в TypeScript?

Категория: Процессы

### Краткий ответ

**Как работает type guards в TypeScript?**  
- механизм сужения типа во время выполнения  
- проверяет тип или свойства значения  
- используется в условных операторах (if, instanceof, typeof)  
- позволяет безопасно обращаться к свойствам/методам конкретного типа  
- поддерживает пользовательские функции-предикаты (is Type)  
- улучшает автодополнение и предотвращает ошибки типов  
- ключевой для работы с union и intersection типами

### Подробный ответ
#### Основной ответ
**Type guards** в TypeScript — это механизмы, позволяющие во время выполнения программы сужать типы переменных, предоставляя компилятору информацию о точном типе в определённых ветках кода. Благодаря им TypeScript может безопасно работать с объединениями типов и обеспечивать проверку типов в разных условиях, предотвращая ошибки.

#### Ключевые моменты
- **Операторы `typeof` и `instanceof`** — встроенные type guards для примитивных типов и классов соответственно. Например, `typeof x === "string"` гарантирует, что внутри этого блока `x` — строка.
- **Пользовательские type guards** — это функции, возвращающие `x is Type`, где `x` — параметр функции. Они дают возможность описать сложные логики сужения типов. Например:
  ```ts
  function isUser(obj: any): obj is User {
    return obj && typeof obj.name === 'string';
  }
  ```
- Type guards работают на уровне **статического анализа**, направляя компилятор, но выполняются во время исполнения.
- Они решают проблему сдержек типа **union types**, когда переменная может иметь несколько разных типов.

#### Практический контекст
В React 18+ и сложных TypeScript-проектах user-defined type guards часто применяются для проверки API-ответов, когда приходят union-объекты с разными структурами. В Node.js проектах type guards помогают учитывать особенности разных типов ошибок, обеспечивая типобезопасный доступ к свойствам конкретных классов ошибок.

Таким образом, type guards — ключевой инструмент для безопасного и выразительного кода на TypeScript.
