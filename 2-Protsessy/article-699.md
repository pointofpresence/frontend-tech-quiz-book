# Что такое list comprehension?

Категория: Процессы

### Краткий ответ

**Что такое list comprehension?**  
- синтаксический сахар Python  
- компактный способ создания списков  
- основан на цикле и условии в одной строке  
- улучшает читаемость и сокращает код  
- выполняется быстрее обычного цикла for  
- поддерживает вложенные циклы и фильтры  
- применяется для трансформаций и фильтраций коллекций

### Подробный ответ
#### Основной ответ
**List comprehension** — это компактный и выразительный способ создания списков в Python, позволяющий генерировать новый список, применяя выражение к каждому элементу итерируемой коллекции или фильтруя элементы по условию. Он заменяет традиционные циклы for и выражения для построения списков, делая код более читаемым и лаконичным.

#### Ключевые моменты
- **Синтаксис** выглядит как `[выражение for элемент in итерация if условие]`, где `if` — необязательный фильтр.
- Позволяет сочетать **итерации и условия** в одной строке, уменьшая количество кода и повышая выразительность.
- За счёт оптимизации CPython list comprehension зачастую работает быстрее эквивалентных циклов с append.
- Может использоваться не только для списков, но и для генераторов (`generator expressions`) и других структур через синтаксис с круглыми скобками.

#### Практический контекст
В реальных проектах list comprehensions применяются для быстрого преобразования данных, например, конвертации списка строк в числа, фильтрации элементов или агрегации. В Python 3.8+ list comprehension использую даже в однострочных lambda-функциях для чистого и эффективного кода. Например:  
```python
squared = [x**2 for x in range(10) if x % 2 == 0]
```
создаст список квадратов чётных чисел от 0 до 9 с высокопроизводительной и понятной реализацией.
