# Что такое оператор EXPLAIN ANALYZE и для чего он используется?

Категория: Технологии

### Краткий ответ

**Что такое оператор EXPLAIN ANALYZE и для чего он используется?**

- инструмент **профилирования и анализа** SQL-запросов  
- сочетает планы выполнения (EXPLAIN) с реальным временем выполнения (ANALYZE)  
- показывает **физический план** запроса, выбор индексов, методы сканирования  
- измеряет время на каждый шаг, количество строк, использование ресурсов  
- помогает выявлять узкие места и оптимизировать запросы  
- крайне полезен при **тюнинге производительности** баз данных  
- поддерживается в основных СУБД: PostgreSQL, MySQL (EXPLAIN ANALYZE в PostgreSQL более информативен)

### Подробный ответ
#### Основной ответ
**Оператор EXPLAIN ANALYZE** в системах управления базами данных, таких как PostgreSQL, используется для анализа и диагностики выполнения SQL-запросов. Он не только показывает план выполнения запроса, но и запускает сам запрос, предоставляя реальные статистические данные — время выполнения каждой операции, количество обработанных строк и затраты ресурсов.

#### Ключевые моменты
- **Показывает фактический план выполнения** запроса, включая операции сканирования, соединений, сортировок и агрегаций с указанием их порядка и структуры.
- **Измеряет реальные показатели** (время, количество строк), что отличается от оператора EXPLAIN без ANALYZE, который дает только прогнозируемую оценку плана.
- **Помогает выявлять "узкие места"** в запросах: например, плохо подобранные индексы, дорогие соединения, избыточные сканы или операции сортировки.
- Используется для **оптимизации производительности** запросов — на основе вывода EXPLAIN ANALYZE можно принимать решения по добавлению индексов, изменению запросов или конфигурации БД.
- В PostgreSQL важный нюанс — оператор **EXPLAIN ANALYZE запускает запрос полностью**, поэтому при больших изменениях он может быть затратным.

#### Практический контекст
Типично используется при оптимизации сложных запросов в PostgreSQL 14+ для снижения времени отклика — например, при терапии медленных отчетов или запросов OLAP. Анализ данных EXPLAIN ANALYZE часто выполняют вместе с утилитами вроде **pgAdmin** или визуализаторами планов, чтобы быстрее находить проблемы в логике запросов и структуре данных.
