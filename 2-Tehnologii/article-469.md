# Как преобразовать целое число (int32) в строку без стандартных функций (используя руны)?

Категория: Технологии

### Краткий ответ

**Преобразование int32 в строку без стандартных функций через руны**

- низкоуровневое преобразование числа в **последовательность цифр**
- берём знак числа, если отрицательное — сохраняем отдельно
- выделяем последовательно **остатки от деления на 10** (число % 10)
- преобразуем каждую цифру в соответствующую **руну ('0' + digit)**
- собираем руны в срез/массив в обратном порядке
- разворачиваем собранный срез для правильного порядка вывода
- при отрицательном числе — вставляем руну минуса в начало
- итог — строка из рун, выражающая число

**Суть:** ручная реализация конвертации через арифметику и руны вместо готовых `strconv.Itoa` или `fmt.Sprintf`.

### Подробный ответ
#### Основной ответ
Преобразование **int32** в строку без использования стандартных функций, таких как `strconv.Itoa` или `fmt.Sprintf`, можно реализовать вручную, ориентируясь на арифметические операции с цифрами числа и последующее формирование строки. Для создания строки в Go, используя руны (runes), мы можем конвертировать каждую цифру в соответствующий символ Unicode, так как руна — это тип `int32` для представления символов.

Основная идея — получить знак числа, обработать отдельно отрицательный случай, затем в цикле извлечь последние цифры через операцию взятия по модулю (`% 10`), преобразовать их в руны, сдвинуть к символам `'0'` (код 48 в Unicode) и собрать в срез руин. В конце поможет обращение среза, чтобы получить корректный порядок цифр.

#### Ключевые моменты
- **Обработка отрицательных чисел**: при отрицательном числе нужно запомнить знак и работать с модулем.
- **Преобразование цифры в руну**: каждая цифра `d` переводится в символ `'0' + d`, где `'0'` это rune с кодом 48.
- **Переворот среза рун**: так как извлечение цифр происходит с конца числа (младшие цифры), результат нужно перевернуть.
- **Преобразование рун в строку**: строка в Go — это срез байт или руны. Можно создать строку непосредственно из среза рун.

#### Практический контекст
Такой подход полезен, если необходимо контролировать процесс конверсии и минимизировать зависимости от внешних библиотек, например, в системах с ограниченными библиотеками или при написании собственных низкоуровневых сериализаторов. Также этот алгоритм часто используется для обучения и понимания механики преобразования чисел в строку.

---

Пример реализации (без использования `strconv` и `fmt`):

```go
func Int32ToString(n int32) string {
    if n == 0 {
        return "0"
    }

    var runes []rune
    negative := false
    if n < 0 {
        negative = true
        n = -n
    }

    for n > 0 {
        digit := n % 10
        runes = append(runes, rune('0'+digit))
        n /= 10
    }

    if negative {
        runes = append(runes, '-')
    }

    // Обращаем срез руны
    for i, j := 0, len(runes)-1; i < j; i, j = i+1, j-1 {
        runes[i], runes[j] = runes[j], runes[i]
    }

    return string(runes)
}
```

Такой код покажет собеседнику, что вы понимаете внутренние детали и умеете обходиться без стандартных вспомогательных функций, используя руны как представление символов.
