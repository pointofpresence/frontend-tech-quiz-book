# Что такое портал в React?

### Категории

UI, Frontend, Web, Components, React

### Краткий ответ

**Что такое портал в React?**

- React портал — способ **рендерить компоненты** вне основного DOM-дерева  
- используется для элементов с особыми требованиями к позиционированию (модалки, тултипы)  
- создаётся через ReactDOM.createPortal(child, container)  
- **сохраняет контекст React**, несмотря на разное расположение в DOM  
- облегчает управление слоями и стилями, избегая конфликтов CSS  
- улучшает UX, позволяя вывести элемент над другими без изменения структуры приложения  
- применяется для **модальных окон, всплывающих уведомлений, оверлеев**

### Подробный ответ
#### Основной ответ
**Портал в React** — это механизм, позволяющий рендерить дочерние элементы компонента вне текущей иерархии DOM, куда обычно монтируется React-приложение. Это особенно полезно, когда нужно разместить UI-элемент, например, модальное окно или тултип, на другом уровне DOM, чтобы избежать проблем с наложением стилей или z-index.

#### Ключевые моменты
- Порталы создаются с помощью метода **ReactDOM.createPortal(children, container)**, где `children` — JSX для рендеринга, а `container` — DOM-узел, в который будет вставлен этот JSX.
- Важно, что несмотря на то, что визуально элементы переносятся в другое место DOM, React компоненты сохраняют свою логику, состояние и контекст, как если бы они были в исходной иерархии.
- Порталы решают задачи, связанные с **выходом за рамки текущего родителя в DOM**, например, для создания всплывающих окон, dropdown-меню или overlay’ев, где важно управление z-index и визуальной модальностью интерфейса.

#### Практический контекст
В крупных приложениях React 18+ порталы используются для создания **модальных окон** с управлением фокусом и анимациями, а также для наложения интерфейсов поверх основного контента, сохраняя при этом логику React и упрощая контроль над CSS и событиями вне основной части DOM.
