# В чём различие между методами HTTP: POST, PUT и PATCH?

### Категории

Methods, API, Networking, HTTP, Testing, Network, REST

### Краткий ответ

**Различия между HTTP методами POST, PUT и PATCH**  
- HTTP методы для **модификации ресурсов**  
- **POST**: создаёт новый ресурс, **неидемпотентен** (много разных результатов)  
- **PUT**: заменяет весь ресурс целиком, **идемпотентен** (повтор вызывает тот же эффект)  
- **PATCH**: частично обновляет ресурс, **идемпотентность зависит от реализации**  
- POST часто для создания, PUT и PATCH — для обновления  
- PUT требует полный объект, PATCH — только изменённые поля  
- Выбор зависит от задачи: создание (POST), полное обновление (PUT), частичное обновление (PATCH)

### Подробный ответ
#### Основной ответ
Методы **HTTP POST**, **PUT** и **PATCH** служат для создания и изменения ресурсов, но различаются по семантике и поведению. **POST** обычно используется для создания новых ресурсов или выполнения действий, **PUT** — для полного замещения существующего ресурса, а **PATCH** — для частичного обновления.

#### Ключевые моменты
- **POST**: Отправляет данные на сервер для создания ресурса или запуска операции. Действие не идемпотентно, т.е. повторный вызов может создать дубликаты. Например, создание пользователя — POST /users.
- **PUT**: Заменяет весь ресурс по указанному URI новым представлением. Идемпотентен — повторный вызов с одними и теми же данными приводит к одному результату. Если ресурс не существует, иногда создаётся. Например, замена профиля пользователя — PUT /users/123.
- **PATCH**: Вносит частичные изменения в ресурс — обновляет только указанные поля, не затрагивая остальные. Обычно идемпотентен (зависит от реализации). Более эффективен при больших ресурсах и частичных обновлениях.

#### Практический контекст
В REST API с React 18 и Node.js часто используют POST для создания, PUT для полной замены объекта, а PATCH — для изменения отдельного поля, например, обновление статуса задачи. Поддержка идемпотентности PUT и PATCH упрощает обработку сетевых повторов и повышает надёжность.
