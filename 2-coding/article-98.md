# Как работают методы indexOf() и lastIndexOf() в JavaScript?

### Категории

Coding, JavaScript, String methods, Event loop, Debugging, TypeScript, Algorithmic complexity

### Краткий ответ

**Как работают методы indexOf() и lastIndexOf() в JavaScript?**

- строковые методы поиска подстроки в JS
- **indexOf()** ищет позицию первого совпадения слева направо
- **lastIndexOf()** ищет позицию последнего совпадения справа налево
- возвращают индекс первого символа или **-1**, если нет совпадения
- работают с опциональным параметром для начала поиска (startIndex)
- чувствительны к регистру
- применимы для строк и массивов (с аналогичным поведением)

### Подробный ответ
#### Основной ответ
Методы **indexOf()** и **lastIndexOf()** в JavaScript используются для поиска позиции подстроки внутри строки. Метод **indexOf()** возвращает индекс первого вхождения указанного значения, а **lastIndexOf()** — индекс последнего вхождения. Если значение не найдено, оба метода возвращают `-1`. Они чувствительны к регистру и работают с простым поиском по вхождению подстроки.

#### Ключевые моменты
- **Поиск слева направо**: `indexOf()` начинает сканирование с начала строки и возвращает индекс первого совпадения, тогда как `lastIndexOf()` ищет с конца строки, возвращая последний индекс вхождения.
- **Параметры**: оба принимают два аргумента — строку для поиска и необязательный индекс, с которого начинается поиск (`fromIndex` для `indexOf`, `fromIndex` с конца для `lastIndexOf`).
- **Производительность**: оба метода работают за время, пропорциональное длине строки, то есть `O(n)`. Для очень длинных строк или повторных поисков имеет смысл использовать другие подходы, например, регэкспы или индексы.
- **Типы данных**: методы определены у прототипа String, но аналогичные версии существуют и у массивов с похожим поведением.

#### Практический контекст
В веб-разработке чаще всего `indexOf()` применяют для проверки наличия подстроки (например, `if (str.indexOf('test') !== -1)`), а `lastIndexOf()` — для поиска последнего разделителя пути, чтобы выделить имя файла или расширение. В современных проектах иногда заменяют на методы `includes()` или `String.prototype.search()` с регулярками для более сложных задач, однако `indexOf()` остаётся самым оптимальным и простым методом для классического поиска подстроки.
