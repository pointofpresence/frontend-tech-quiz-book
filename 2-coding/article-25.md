# Map. В чем особенность? Какие свойства и методы есть? Как перебрать?

### Категории

Algorithm complexity, TypeScript, Debugging, Console output, Coding, Iterables, Methods, Data structures, Event loop

### Краткий ответ

**Map: особенности, свойства, методы, перебор**

- структура данных для хранения **пар ключ-значение**
- ключи могут быть любого типа, включая объекты
- порядок элементов сохраняется по вставке
- основные методы:  
  - `set(key, value)` — добавить/обновить пару  
  - `get(key)` — получить значение по ключу  
  - `has(key)` — проверить наличие ключа  
  - `delete(key)` — удалить по ключу  
  - `clear()` — очистить все элементы  
  - `size` — количество элементов  
- перебор с помощью:  
  - `for...of map` (итерация по `[key, value]`)  
  - методов `map.keys()`, `map.values()`, `map.entries()`  
- используется для быстрой итерируемой **ассоциативной коллекции** с преимуществом над объектом в гибкости ключей и порядке

### Подробный ответ
#### Основной ответ
**Map** — это встроенная структура данных в JavaScript, появившаяся в стандарте ES6, которая хранит пары ключ-значение и сохраняет порядок вставки элементов. В отличие от обычных объектов, ключами в Map могут быть любые типы данных, включая объекты, функции и примитивы. Это делает Map более универсальной и предсказуемой с точки зрения поведения и итерирования.

#### Ключевые моменты
- **Ключи любого типа**: Map не приводит ключи к строкам, в отличие от объектов, поэтому `map.set({}, 'value')` и `map.get({})` будут разными элементами.
- **Порядок элементов**: перебор элементов Map происходит в порядке их вставки — это важно для предсказуемого итерационного поведения.
- **Основные методы**:
  - `set(key, value)` — добавляет или обновляет пару ключ-значение
  - `get(key)` — возвращает значение по ключу или `undefined`, если ключа нет
  - `has(key)` — проверяет наличие ключа, возвращает булево
  - `delete(key)` — удаляет ключ и связанное с ним значение
  - `clear()` — очищает все элементы в Map
- **Свойства**:
  - `size` — количество пар ключ-значение в Map
- **Перебор**:
  - Можно перебирать Map с помощью методов `map.forEach(callback)` или используя `for...of` с деструктуризацией:  
    ```js
    for (const [key, value] of map) {
      // действия с key и value
    }
    ```
  - Методы `map.keys()` (получить итератор ключей), `map.values()` (значения), `map.entries()` (пары) полезны для итераций.

#### Практический контекст
Map широко используют, когда нужна эффективная итерируемая коллекция с произвольными ключами — например, кеши, кеширование результатов функций, хранение связей между объектами. Практично использовать с React 18 и Node.js 16+ для хранения связанного состояния без преобразования ключей в строки и с предсказуемым порядком.
