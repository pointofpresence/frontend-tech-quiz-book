# Как вернуть первые два подряд идущих равных элемента из массива?

### Категории

Iteration, Algorithm, Coding, Data structures, Array

### Краткий ответ

**Как вернуть первые два подряд идущих равных элемента из массива?**

- Работа с массивом, поиск последовательности  
- Проход по элементам с проверкой соседних значений  
- Сравниваем элемент с следующим: если равны — возвращаем пару  
- Время работы O(n), где n — длина массива  
- При отсутствии пары — возвращаем null или пустой результат  
- Удобно реализовать циклом for с проверкой i и i+1  
- Применимо в задачах поиска шаблонов и оптимизации данных

### Подробный ответ
#### Основной ответ
Чтобы вернуть первые два подряд идущих равных элемента из массива, нужно пройтись по массиву и сравнивать каждый элемент с следующим. Как только найдёте пару совпадающих подряд элементов, вернуть их. Это стандартная задача линейного поиска по массиву с сравнением соседних элементов.

#### Ключевые моменты
- Проходим массив с индексом от 0 до n-2, сравнивая `arr[i]` с `arr[i+1]`.
- При первом совпадении возвращаем пару, например `[arr[i], arr[i+1]]`.
- Если таких пар нет — возвращаем `null` или пустой результат.
- Сложность алгоритма — O(n), где n — длина массива. Это оптимально, т.к. нужно проверить пары подряд.

#### Практический контекст
В реальных задачах такой подход может использоваться для детекции дубликатов подряд (например, при очистке данных, обработке потоков событий или выявлении повтора в логах). Также удобно для валидации пользовательского ввода, где повтор символов или значений недопустим. В JavaScript и Python решение часто занимает 3-5 строк, что идеально для интервью.
