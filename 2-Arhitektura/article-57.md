# Реализовать функцию map.

Категория: Архитектура

### Краткий ответ

**Реализация функции map**

- функция **map** трансформирует массив
- применяет переданную функцию ко всем элементам
- возвращает новый массив с преобразованными значениями
- не изменяет исходный массив (иммутабельность)
- итерация с помощью цикла for или метода forEach
- имеет сложность **O(n)**, n — длина массива
- используется для **чистой функциональной трансформации данных**

Пример на JS:

```js
function map(arr, fn) {
  const result = [];
  for (let i = 0; i < arr.length; i++) {
    result.push(fn(arr[i], i, arr));
  }
  return result;
}
```

Это классическая версия, которая точно повторяет нативный Array.prototype.map по сигнатуре и поведению.

### Подробный ответ
#### Основной ответ
Функция **map** — это метод в функциональном программировании и в языках программирования, который применяется к коллекции (массиву, списку и т.п.) и возвращает новый массив, где каждый элемент — результат применения переданной функции к соответствующему элементу исходного массива. Проще говоря, map трансформирует данные, не изменяя исходную структуру.

#### Ключевые моменты
- **Иммутабельность**: исходный массив не модифицируется, функция map возвращает новый массив.
- **Функция высшего порядка**: принимает другую функцию в качестве аргумента, которая определяет логику преобразования каждого элемента.
- **Итерация с сохранением индекса**: часто функция принимает элемент и его индекс, что расширяет возможности трансформации.
- При реализации важно учитывать, что map — ленивый или жадный (в языках вроде JavaScript — жадный), и корректно обрабатывать пустые или null-значения.

#### Практический контекст
В JavaScript, начиная с ECMAScript 5+, map встроена как метод Array.prototype. Например:  
```js
const numbers = [1, 2, 3];
const doubled = numbers.map(x => x * 2); // [2, 4, 6]
```
При реализации с нуля в JS можно сделать так:
```js
function map(array, fn) {
    const result = [];
    for (let i = 0; i < array.length; i++) {
        result.push(fn(array[i], i, array));
    }
    return result;
}
```
В TypeScript добавим типизацию для безопасности. В реальных проектах map используют для трансформации данных перед рендерингом UI или обработкой массивов без мутаций, что повышает читаемость и надежность кода.

Важно понимать и различать map и forEach — map возвращает новый массив, forEach просто перебирает без возврата значений.
