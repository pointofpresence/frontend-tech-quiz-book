# Приведите примеры агрегатных функций.

### Категории

Data processing, Database, Query, Aggregation, Technical

### Краткий ответ

**Примеры агрегатных функций**

- работают с набором данных, возвращают одно значение  
- **SUM()** — сумма значений  
- **AVG()** — среднее арифметическое  
- **COUNT()** — количество элементов  
- **MAX()** — максимальное значение  
- **MIN()** — минимальное значение  
- часто используются в SQL для **группировки** и **вычислений на уровне базы**

### Подробный ответ
#### Основной ответ
**Агрегатные функции** — это функции, которые принимают набор значений (обычно из столбца таблицы базы данных) и возвращают одно сводное значение. Они широко применяются для анализа данных и извлечения статистической информации на уровне групп или всей таблицы.

#### Ключевые моменты
- **SUM()** — возвращает сумму всех значений в столбце, часто применяется для подсчёта доходов, расходов и других метрик.
- **COUNT()** — подсчитывает количество строк или значений, полезна для определения объёма данных.
- **AVG()** — вычисляет среднее арифметическое числовых значений.
- **MIN() и MAX()** — находят минимальное и максимальное значение в выборке, соответственно.
- Некоторые системы, например PostgreSQL 14+, дополнительно поддерживают функции как **STRING_AGG()** для объединения строк и **ARRAY_AGG()** для сбора значений в массив.
- Агрегатные функции обычно используются в сочетании с оператором **GROUP BY** для группировки данных.

#### Практический контекст
В реальных проектах агрегатные функции — основа отчетности и аналитики: например, чтобы посчитать средний чек покупок, общую выручку по дням или максимальный рейтинг товара. Использование их в запросах к PostgreSQL, MySQL, SQL Server существенно ускоряет обработку данных без необходимости извлекать их на стороне приложения.
