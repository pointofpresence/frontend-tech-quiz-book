# Как найти все элементы на странице?

### Категории

DOM manipulation, Algorithm, String, Frontend, Coding challenge, Technical

### Краткий ответ

**Как найти все элементы на странице?**

- DOM — структура HTML в дереве элементов  
- Использовать `document.querySelectorAll('*')` — селектор всех элементов  
- Возвращает **NodeList** со всеми элементами в документе  
- Итерируем для обработки/фильтрации нужных узлов  
- Можно использовать `document.getElementsByTagName('*')` — HTMLCollection, устаревший API  
- Применение: анализ, модификация всей страницы, сбор статистики  
- Быстрый перебор с минимальными затратами ресурсов браузера

### Подробный ответ
#### Основной ответ
Чтобы найти все элементы на странице в браузере, обычно используется метод API DOM, который позволяет перебрать или выбрать узлы документа. Самый простой способ — использовать метод **document.querySelectorAll('*')**: он возвращает список всех элементов (всех тегов) на текущей странице. Альтернативно можно воспользоваться **document.getElementsByTagName('*')**, однако querySelectorAll возвращает статический NodeList, тогда как getElementsByTagName — живую коллекцию.

#### Ключевые моменты
- **querySelectorAll('*')** — возвращает все элементы независимо от типа с помощью CSS-селектора `*`, удобен благодаря поддержке сложных селекторов и совместимости с современными браузерами (включая React 18+ и Chrome 110+).
- **getElementsByTagName('*')** — более старый метод, возвращает живую коллекцию, что может влиять на производительность при изменении DOM.
- Эффективность и потребление памяти важно учитывать, если страница большая — получение всех элементов может быть дорогостоящей операцией.

#### Практический контекст
Обычно это используется для обхода DOM (например, при написании скриптов автоматизации, парсинга или в end-to-end тестах). В инструментах типа Puppeteer или Cypress выбирают все элементы, чтобы применить фильтрацию по атрибутам или классам. В продакшене лучше избегать частого обращения ко всем элементам из-за затрат на производительность.
