# Что такое перечисления (enumerations) в Swift?

### Категории

Programming concepts, Data types, Enumerations, Technical

### Краткий ответ

**Что такое перечисления (enumerations) в Swift?**  
- структура данных для **группировки связанных значений**  
- могут содержать **связанные значения** (associated values) разных типов  
- поддерживают **методы и свойства**, работают как полноценные типы  
- позволяют **безопасно управлять вариантами** в коде (switch, pattern matching)  
- поддерживают **рекурсивные перечисления** для сложных структур  
- облегчают **читаемость и безопасность** кода через строгую типизацию  
- широко применяются для **описания состояний, вариантов и событий** в приложениях Swift

### Подробный ответ
#### Основной ответ
**Перечисления (enumerations или enums)** в Swift — это мощный тип данных, который позволяет определить набор связанных значений как единый тип. В отличие от многих языков, где енумы просто именуют набор констант, в Swift перечисления могут иметь связанные значения, методы и даже вычисляемые свойства, что значительно расширяет их функциональность.

#### Ключевые моменты
- **Ассоциированные значения**: каждая "ветка" перечисления может содержать данные различных типов, которые хранятся вместе с самим кейсом. Например, `case success(Int)` и `case failure(String)` в одном enum удобны для обработки результатов.
- **Вычисляемые свойства и методы**: к enum можно добавлять функции и свойства, что позволяет инкапсулировать логику, связанную с этим типом, прямо внутри него.
- **Безопасность типов**: компилятор требует обязательного exhaustive pattern matching, что повышает надежность и облегчает поддержку кода.
- **Enum с raw values**: можно задать базовые значения (`Int`, `String` и др.), которые автоматически присваиваются кейсам, что удобно для коммуникации с C API или сериализации.

#### Практический контекст
В реальных проектах Swift enums широко применяются для управления состояниями, обработки ошибок (`Result` — стандартный тип с enum), навигации по MVVM или Redux-подобным архитектурам благодаря возможности описывать события и действия с различными данными в одном типе. В Swift 5+ это особенно удобно благодаря улучшенной работе с сопоставлением с образцом.
