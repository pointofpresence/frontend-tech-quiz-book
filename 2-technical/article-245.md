# Какими инструменты профилирования знаешь?

### Категории

Profiling, Game dev, Performance, Optimization, Technical

### Краткий ответ

**Инструменты профилирования**

- CPU профилирование: Perf, VTune, VisualVM  
- Память: Valgrind, Heap Profiler, Memory Profiler (Python)  
- Сетевая активность: Wireshark, tcpdump  
- Веб и фронтенд: Chrome DevTools Profiler, Lighthouse  
- Профилирование Go: pprof  
- Важное: выбирать по языку и задаче  
- Цель: выявление узких мест и утечек

### Подробный ответ
#### Основной ответ
Для профилирования приложений я использую широкий набор инструментов, которые зависят от языка программирования и среды исполнения. Например, для **C++** часто применяю **Valgrind** и **Perf** в Linux, для **Java** — **VisualVM**, **JProfiler** и встроенный **Java Flight Recorder**. В веб- и Node.js-проектах применяю **Chrome DevTools** и **Node.js Profiler**. Для анализа производительности и мониторинга микросервисов — **Prometheus** и **Grafana**.

#### Ключевые моменты
- **Valgrind** — мощный инструмент для выявления утечек памяти и профилирования CPU, особенно полезен на низком уровне, но с заметным overhead.
- **Perf** предоставляет детальные hardware-события и профилирование в Linux с низкой задержкой, отлично подходит для анализа системных характеристик на продакшене.
- В экосистеме Java **Java Flight Recorder** (начиная с JDK 11+) предлагает низкоуровневый мониторинг с минимальным влиянием на производительность.
- Для фронтенда **Chrome DevTools Performance Panel** позволяет визуализировать загрузку CPU, рендеринг, и выявлять холодные участки кода.
- Инструменты вроде **Flamegraph** и **Speedscope** помогают визуализировать профилирование в виде удобных flame-графиков.

#### Практический контекст
В реальных проектах я комбинирую инструменты: например, для продакшен-среды использую lightweight профилирование на основе множества метрик через Prometheus, а при локальной оптимизации — глубокое профилирование через Valgrind или JFR. В React-приложениях регулярно использую React Profiler в Chrome для оценки времени рендера и выявления лишних перерисовок.
