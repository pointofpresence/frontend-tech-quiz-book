# Перечислите основные методы массивов.

**Основные методы массивов в JavaScript**

- **push()** — добавляет элементы в конец, **изменяет длину**
- **pop()** — удаляет последний элемент, возвращает его
- **shift()** — удаляет первый элемент, сдвигает остальные влево
- **unshift()** — добавляет элементы в начало массива
- **map()** — трансформирует элементы, возвращает новый массив
- **filter()** — фильтрует элементы по условию, новый массив
- **reduce()** — сводит массив к одному значению по функции
- **forEach()** — выполняет функцию для каждого элемента, без возврата
- **slice()** — возвращает копию части массива, не изменяет оригинал
- **splice()** — изменяет содержимое, добавляет/удаляет элементы
- **find()** — возвращает первый элемент, подходящий под условие
- **some() / every()** — проверяет наличие/полное соответствие условию, возвращает boolean
- **sort()** — сортирует массив **in-place**, можно с кастомной функцией
- **concat()** — объединяет массивы без изменения исходных
- **includes()** — проверяет наличие элемента, возвращает boolean

Эти методы покрывают основные операции: модификация, итерация, фильтрация и трансформация. Владение ими обязательно для эффективной работы с данными в JS.

# Подробный ответ
## Основной ответ
В JavaScript массивы обладают богатым набором встроенных **методов** для работы с данными — они позволяют изменять, перебирать, фильтровать и трансформировать элементы массива. Основные методы можно разделить на несколько групп по назначению: перебор, изменение, поиск и преобразование.

## Ключевые методы массивов
- **forEach(callback)** — перебирает каждый элемент массива, выполняя функцию обратного вызова, не возвращает новый массив (side-effect).
- **map(callback)** — создает новый массив с результатами вызова callback для каждого элемента, не меняет оригинал.
- **filter(callback)** — возвращает новый массив с элементами, для которых callback вернул true.
- **reduce(callback, initialValue)** — сводит массив к одному значению, аккумулируя результат через callback.
- **find(callback)** — возвращает первый элемент, удовлетворяющий условию, либо undefined.
- **some(callback)** и **every(callback)** — проверяют, удовлетворяют ли элементы условию (хотя бы один или все).
- **push(...items)** и **pop()** — добавляют и удаляют элементы в конец массива, изменяя оригинал.
- **shift()** и **unshift(...items)** — соответственно удаляют и добавляют элементы в начало массива.
- **splice(start, deleteCount, ...items)** — универсальный метод для удаления, замены и вставки элементов в произвольное место массива.
- **slice(start, end)** — возвращает новый массив с копией части оригинального массива, не изменяя его.
- **concat(...arrays)** — объединяет массивы, возвращая новый.
- **sort(compareFunction)** — сортирует массив на месте, возможна передача кастомной функции сравнения.
- **reverse()** — инвертирует порядок элементов в оригинальном массиве.
- **indexOf(value)** и **lastIndexOf(value)** — ищут позицию элемента в массиве.

## Практический контекст
Эти методы — базовый инструментарий при повседневной разработке на JavaScript, особенно в React 18+ и Node.js. Например, map и filter часто применяют для отображения списков или фильтрации данных по условиям, reduce — для агрегаций и свёрток. Знание особенностей мутабельных (push, splice) и немутабельных (map, filter) операций критично для корректной работы с состоянием и данными.
