# Как создать простое модальное окно на чистом JavaScript без использования библиотек?

### Категории

Behavioral, JavaScript, Promises, Async, Templating, UI, Frontend

### Краткий ответ

**Как создать простое модальное окно на чистом JavaScript**

- DOM-элементы: создаём контейнер, фон и контент  
- CSS: фиксированное позиционирование, затемнение фона, центрирование  
- JS: добавляем элементы в body, управляем видимостью через style.display  
- Обработчики клика: закрытие по фону и кнопке "закрыть"  
- Фокус и скролл: блокируем скролл при открытии модального окна  
- Простота и независимость: полностью без библиотек, легковесно  
- Практика: подходит для уведомлений, подтверждений, форм ввода

### Подробный ответ
#### Основной ответ
Создать простое **модальное окно** на чистом JavaScript можно через динамическое добавление в DOM элементов, которые имитируют фон затемнения и само окно, а также управление их видимостью и событиями взаимодействия. Важно реализовать базовые UX-аспекты: блокировку фона, закрытие окна по клику вне модалки или по кнопке, и фокусировку на окне для доступности.

#### Ключевые моменты
- **Структура HTML/CSS**: создаем оверлей (полупрозрачный фон) и контейнер модального окна с контентом. Оверлей занимает всю видимую область, фиксирован позиционирован, чтобы блокировать взаимодействие с остальной страницей.
- **Создание и управление через JavaScript**: с помощью `document.createElement` добавляем элементы в DOM, навешиваем обработчики событий (например, по клику на фон или кнопку закрытия), чтобы управлять видимостью модалки через стиль `display` или `visibility`.
- **Фокус и доступность**: при открытии модального окна стоит программно переводить фокус на окно или первый интерактивный элемент, а при закрытии возвращать его назад к исходному. Также желательно предотвратить прокрутку фонового контента (`overflow: hidden` на body).

#### Практический контекст
Такой подход типично используют для легких проектов или компонентов без подключенных фреймворков. На крупных проектах обычно применяют готовые UI-библиотеки (например, Bootstrap модалки или кастомные React-компоненты) для надежной работы с а11y и анимациями. Но понимание "с нуля" помогает лучше разобраться в механике.

---
Пример кода для создания простого модального окна:

```js
// Создаем элементы
const overlay = document.createElement('div');
overlay.style = `
  position: fixed; top: 0; left: 0; width: 100vw; height: 100vh;
  background: rgba(0,0,0,0.5); display: flex; justify-content: center;
  align-items: center; z-index: 9999;
`;

const modal = document.createElement('div');
modal.style = `
  background: white; padding: 20px; border-radius: 8px;
  max-width: 400px; width: 100%;
`;

modal.innerHTML = `<p>Это простое модальное окно</p><button id="closeBtn">Закрыть</button>`;

// Добавляем в DOM
overlay.appendChild(modal);
document.body.appendChild(overlay);

// Убираем скролл на фоне
document.body.style.overflow = 'hidden';

// Закрытие по кнопке
document.getElementById('closeBtn').onclick = () => {
  document.body.removeChild(overlay);
  document.body.style.overflow = '';
};

// Закрытие по клику вне модалки
overlay.onclick = (e) => {
  if (e.target === overlay) {
    document.body.removeChild(overlay);
    document.body.style.overflow = '';
  }
};
```

Таким образом создается простой, функциональный модал, который можно расширять по необходимости.
