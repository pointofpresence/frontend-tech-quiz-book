# Что такое Web Vitals?

### Категории

Behavioral, Containerization, Metrics, Async, Performance, Optimization, Frontend

### Краткий ответ

**Что такое Web Vitals?**

- метрики для оценки **юзабилити и производительности** сайтов  
- разработаны Google под UX-фокус  
- ключевые показатели:  
  - **LCP** (Largest Contentful Paint) — загрузка основного контента  
  - **FID** (First Input Delay) — задержка отклика на взаимодействие  
  - **CLS** (Cumulative Layout Shift) — стабильность визуального контента  
- измеряют **скорость**, **интерактивность**, и **визуальную стабильность**  
- помогают оптимизировать сайт под реальные пользовательские сценарии  
- используются в SEO и улучшении показателей конверсии  
- доступны в инструментах: Chrome DevTools, Lighthouse, PageSpeed Insights

### Подробный ответ
#### Основной ответ
**Web Vitals** — это набор метрик, разработанных Google для оценки пользовательского опыта на веб-страницах, фокусируясь на ключевых аспектах производительности и взаимодействия. Цель этих метрик — дать разработчикам стандартизированные показатели, которые отражают реальное восприятие сайта посетителями, и помочь оптимизировать важные параметры загрузки и отклика страниц.

#### Ключевые моменты
- Включают три основные метрики: **Largest Contentful Paint (LCP)**, измеряющий время загрузки основного контента; **First Input Delay (FID)** — задержку первого взаимодействия пользователя с сайтом; и **Cumulative Layout Shift (CLS)** — визуальную стабильность страницы.
- Web Vitals стандартизируют важные для SEO и UX показатели, учитывая реальные пользовательские сценарии, а не только чисто технические (например, время загрузки).
- Google активно продвигает эти метрики, интегрируя их в инструменты мониторинга (PageSpeed Insights, Chrome UX Report) и алгоритмы ранжирования, делая их критически важными для разработчиков и SEO-специалистов.

#### Практический контекст
В React 18+ или Next.js с поддержкой SSR, оптимизация Web Vitals — это приоритет: критически важно уменьшить LCP за счет lazy loading изображений и SSR, снизить FID через оптимизацию JavaScript, а CLS контролировать путем резервирования размеров элементов. Web Vitals часто мониторят в реальном времени через Google Analytics или Lighthouse для постоянного улучшения пользовательского опыта.
