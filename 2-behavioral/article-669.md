# Как решить задачу на группировку с помощью Stream API?

### Категории

Behavioral, API, Java 8, Functional programming, Data processing, Streaming

### Краткий ответ

**Как решить задачу на группировку с помощью Stream API?**

- Stream API — средство для обработки коллекций в Java  
- группировка через метод **Collectors.groupingBy()**  
- принимает функцию классификации (keyMapper)  
- возвращает Map с ключами и списками элементов  
- можно использовать downstream-коллекторы (например, подсчёт, усреднение)  
- эффективно для агрегаций и категоризации данных  
- позволяет писать чистый, декларативный и читаемый код  

Пример:  
```java
Map<Category, List<Item>> grouped = items.stream()
    .collect(Collectors.groupingBy(Item::getCategory));
```

### Подробный ответ
#### Основной ответ
Для решения задачи группировки с помощью **Stream API** в Java используется метод **Collectors.groupingBy()**, который позволяет сгруппировать элементы коллекции по заданному признаку, представленному функцией классификации (classifier function). Этот метод возвращает `Map`, где ключами выступают результаты классификатора, а значениями — списки соответствующих элементов.

Процесс выглядит так: сначала из коллекции создаётся стрим, затем применяется операция группировки, передавая функцию, по которой происходит классификация (например, группировка по полю объекта). Это удобный и декларативный способ агрегации данных без необходимости писать циклы вручную.

#### Ключевые моменты
- **Collectors.groupingBy** может принимать не только функцию классификации, но и `Collector` для более сложной агрегации, например, подсчет количества элементов (с помощью `Collectors.counting()`) или суммирование;
- В Java 8-11 это стандартный способ группировки, начиная с Java 12 есть удобное API для additional downstream collectors, позволяющие делать вложенные группировки с несколькими уровнями;
- Группировка возвращает неизменяемую структуру Map при использовании стандартных методов, и можно задать конкретный тип Map (например, `LinkedHashMap`), если важен порядок.

#### Практический контекст
Пример: есть список сотрудников `List<Employee>`, нужно сгруппировать по отделам. Используем:

```java
Map<String, List<Employee>> grouped = employees.stream()
    .collect(Collectors.groupingBy(Employee::getDepartment));
```

Это создаст карту, где ключ — название отдела, а значение — список сотрудников из этого отдела. Для подсчёта количества сотрудников в каждом отделе:

```java
Map<String, Long> countByDept = employees.stream()
    .collect(Collectors.groupingBy(Employee::getDepartment, Collectors.counting()));
```

Такой подход широко используется при агрегации данных из БД или CSV, при обработке логов и т.п. Он позволяет писать декларативный, лаконичный и легко поддерживаемый код.
