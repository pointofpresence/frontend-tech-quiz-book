# Что такое статическая генерация страниц (Static Site Generation, SSG) и в каких случаях она полезна?

### Категории

Behavioral, React, Nested structures, SSG, Static site generation, Frontend, UI components, Forms

### Краткий ответ

**Что такое статическая генерация страниц (Static Site Generation, SSG) и в каких случаях она полезна?**

- веб-рендеринг до запуска (build-time)
- формирует готовые HTML-страницы
- быстрое время загрузки, снижает нагрузку на сервер
- статические сайты изменяются редко
- хорошо для блогов, документации, лендингов
- не подходит для часто меняющегося контента
- улучшает SEO за счёт готового HTML

Итог: SSG оптимален для стабильного, быстро загружаемого контента с SEO-выгодой.

### Подробный ответ
#### Основной ответ
**Статическая генерация страниц (Static Site Generation, SSG)** — это метод построения веб-сайтов, при котором HTML-страницы генерируются заранее на этапе сборки (build time), а не динамически на сервере при каждом запросе. Такая генерация позволяет создавать полностью готовые HTML-файлы, которые затем разворачиваются на CDN или обычном веб-сервере и отдаются пользователям с минимальной задержкой.

#### Ключевые моменты
- **Производительность и масштабируемость**: поскольку страницы уже сгенерированы, серверу не нужно тратить ресурсы на рендеринг при каждом запросе, что снижает нагрузку и увеличивает скорость загрузки (latency может быть ~50ms или ниже по всему миру через CDN).
- **SEO и безопасность**: SSG идеально подходит для контента, который редко меняется — поисковые системы легко индексируют статические HTML, а отсутствие серверной логики снижает поверхность атак.
- **Ограничения**: обновления контента требуют повторной генерации всего сайта или соответствующих страниц, что может быть нетривиально для часто изменяющегося контента.

#### Практический контекст
SSG часто используется для блогов, лендингов, документации и маркетинговых сайтов — например, **Next.js (с поддержкой static export), Gatsby, Hugo и Jekyll** — популярные инструменты для SSG. В проектах с высокой посещаемостью и преимущественно статическим контентом SSG обеспечивает простую инфраструктуру и отличную пользовательскую скорость.
