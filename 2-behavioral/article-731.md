# Различия между append() и extend() для списков.

### Категории

Lists, Behavioral, Python, Data structures, Testing

### Краткий ответ

**Различия между append() и extend() для списков**

- Работа со списками Python  
- **append()** добавляет **один элемент** в конец списка  
- **extend()** добавляет **все элементы** итерируемого объекта  
- append([1,2]) → вложенный список  
- extend([1,2]) → расширяет список элементами  
- append — **одиночная вставка**, extend — **массовая вставка**  
- Практическое: append для элемента, extend для слияния списков или коллекций

### Подробный ответ
#### Основной ответ
Методы **append()** и **extend()** — это стандартные операции для работы со списками в Python, но они выполняют разные функции. **append()** добавляет один элемент в конец списка как отдельный объект, тогда как **extend()** расширяет список, добавляя поэлементно все элементы итерируемого объекта (например, другого списка).

#### Ключевые моменты
- **append(x)** — принимает один аргумент `x` любого типа и добавляет его целиком в конец списка. Если `x` — список, он будет добавлен как вложенный список.
- **extend(iterable)** — принимает итерируемый объект и добавляет каждый элемент этого объекта по отдельности в конец списка, увеличивая его длину на количество элементов iterable.
- Если использовать **append()** с другим списком, длина исходного списка увеличится на 1, а при использовании **extend()** — на длину добавляемого списка.
- По производительности **extend()** обычно быстрее при добавлении нескольких элементов, чем последовательные вызовы **append()** в цикле, поскольку расширяет список одним вызовом.

#### Практический контекст
В реальных задачах **append()** подходит, когда нужно добавить один объект (например, новый элемент или вложенный список), а **extend()** — когда нужно объединить списки или добавить несколько элементов сразу, например, объединение результатов нескольких запросов или парсинга. Знание разницы помогает предотвратить ошибки с вложенностью данных и оптимизировать работу с большими коллекциями.
